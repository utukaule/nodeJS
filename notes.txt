4)  MongoDB basic commands
    
    1.  Create database
    ->  use database_name
    
    2.  Show database
    ->  show dbs
    
    3.  Create Collection?
    ->  db.createCollection("Collection_name")
    
    4.  Delete Collection?
    ->  db.Collection_name.drop()
    
    5.  Delete Database?
    ->  db.dropDatabase()

CRUD Operations in MongoDB
    
    1.  How to insert data collection?
    ->  db.collection_name.insertOne({create obj here})
    ->  eg.
        ->  db.products.insertOne({
            name:"note 7 pro",
            brand:'realme',
            price:10000,
            category:"mobiles"
            })
    
    2.  How to check inserted data?
    ->  db.collection_name.find()
    ->  eg.
        ->  db.products.find()

    3.  How to updata data?
    ->  db.collection_name.updateOne({find_product},{$set:{updata_new_information}})
                                            ^               ^
                                            |               |
                              (product previous data)(Product updated data) 
    ->  eg.
        ->  db.products.updataOne({name:'apple'},{$set:{price:100000}})                

    4.  how to delete data?
    ->  db.collection_name.deleteOne({brand:"apple"})
    ->  eg.
        ->  db.products.deleteOne({brand:"apple"})
